Armônia

Armônia, a very elegant programming language

Armônia is a container & object-oriented programming language that values, most of all:

Elegance.

But also the “three C’s”:

  1. yes to Cleanliness,
  2. most of all, Control,
  3. least of all, Complexity

Armônia favors code that is easy to write, debug, and read, but it doesn't fight the human urge for control

You can think of Armônia as an elegant C++
